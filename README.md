# Hotel Management System

A Hotel Management System developed using Flask.

## Features

- User Registration & Login
- Customer Profile Management
- Hotel Search
- Room Booking
- Booking History
- Admin Dashboard
- Room Management
- Amenities Management
- Complaint Management
- Reviews


## Technologies Used

- Python
- Flask
- SQLAlchemy
- Flask-Login
- Flask-WTF
- SQLite
- HTML
- CSS
- JavaScript

## Installation

### Clone the repository

```bash
git clone https://github.com/raamsaranyak/Hotel-Management-System.git
```

### Go to the project folder

```bash
cd Hotel-Management-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the project

```bash
python run.py
```
